# English Cards Bot

Telegram-бот для изучения слов с использованием карточек и простого алгоритма повторения.
На данный момент реализован MVP:
- добавление слов
- тренировка карточек
- простая логика диалога через FSM

## Технологии

- Go 1.25 — основная логика приложения
- PostgreSQL — хранение данных
- Docker / Docker Compose — контейнеризация
- Telegram Bot API — взаимодействие с Telegram
- FSM (Finite State Machine) — управление состояниями пользователя

## Установка и запуск

 1. Клонировать репозиторий

```bash
git clone https://github.com/<your-username>/english_bot.git
cd english_bot
```

 2. Создать `.env` файл

```bash
make env
```

 3. Запустить PostgreSQL

```bash
make up
```

 4. Применить миграции

```bash
make migrate-up
```

 5. Запустить приложение

```bash
make run
```

---

## Проверка работы

Открой Telegram и отправь боту команду:

```text
/start
```

Если бот отвечает — всё работает корректно.
